# Code of Conduct

## Purpose

This repository documents vulnerabilities disclosed through CERT/CC coordinated vulnerability disclosure (VU#653116). The purpose is to inform affected users and enable informed security decisions.

## Expected Behavior

- Discuss findings factually and cite sources
- Respect the coordinated disclosure process
- Do not post credentials, working exploits, or proof-of-concept code
- Do not post personal information of any individual
- Direct technical questions to the CERT/CC VINCE case record

## Unacceptable Behavior

- Publishing exploitation tools or working attack code
- Sharing credentials, API keys, or connection strings
- Harassment or personal attacks against any party
- Posting personal information (doxxing)
- Using this repository to coordinate attacks against any system

## Scope

This code of conduct applies to all interactions within this repository, including issues, pull requests, and discussions.

## Reporting

Report violations by opening an issue or contacting the researcher directly.

## Enforcement

Violations will result in content removal and, if necessary, blocking.
