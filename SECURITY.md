# Security Policy

## About This Repository

This repository is a vulnerability disclosure, not a software product. It documents findings from CERT/CC VU#653116 / CISA ICSA-26-055-03.

## If You Discover a New Vulnerability in Gardyn Products

**Do not open an issue here.** Instead:

1. **Report to CISA:** [https://www.cisa.gov/report](https://www.cisa.gov/report)
2. **Report to CERT/CC:** [https://www.kb.cert.org/vuls/report/](https://www.kb.cert.org/vuls/report/)
3. **Report to the vendor:** security@mygardyn.com

CISA coordinates ICS/IoT vulnerability disclosures and will assign CVE identifiers.

## If You Find an Error in This Disclosure

Open an issue in this repository with the specific claim and the correct information with a source citation.

## Responsible Disclosure
