# Contributing

## This Is a Disclosure Repository

This repository documents findings from CERT/CC VU#653116 / CISA ICSA-26-055-03. It is not an active software project.

## How to Contribute

### Corrections

If you identify a factual error (incorrect CVSS score, wrong date, misattributed CVE), open an issue with:

- The specific claim that is incorrect
- The correct information with source

### Affected Users

If you are a Gardyn device owner and can confirm or deny a finding's current status on your device, open an issue describing:

- Your firmware version (Settings > Device Info)
- What you observed
- Date of observation

Do **not** include your device ID, account email, or any personal information.

### What Not to Submit

- Working exploits or proof-of-concept code
- Credentials, API keys, or connection strings
- Personal information of any individual
- Speculation without evidence

## Pull Requests

Pull requests for factual corrections are welcome. All changes must include a source citation.
